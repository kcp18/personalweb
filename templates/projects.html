The Prediction of Bitcoin Transactions (ML)

Google Cloud Platform, Apache Spark, and Python (Currenly Working On)

*Note
This project is to predict the number of Bitcoin transactions in the future (by years); it can also be seen as the growth of the Bitcoin Blockchain size. I will push the source code/result to my github page pretty soon.

  • It uses Google Cloud Platform to perform the Machine Learning project.
  • Bitcoin Blockchain Dataset from Google BigQuery (821GB) 
  • Uses Google Dataproc to run the Machine Learning model in the Apache Spark framework. 
  • Runs Linear Regression Model to predict the future number of transactions in Bitcoin
  • PySpark to use Spark’s Machine Learning Library

browserhistory (A PyPI Package)

Firefox, Chrome, and Safari History Datasets

*Note
This module was created for the purpose of promoting the digital wellbeing, analyzing users’ browsing behaviors to help users understand their Internet use; It’s not a hacking tool. If you're concerned with privacy, you can read about the discussion I posted on Reddit .

  • A Python module that extracts browsers history from a user’s local computer into Python objects
  • It can also writes the browsing history data into csv files
  • Supports browsers: Firefox, Google Chrome, and Safari
  • Platforms: Linux, MacOS, and Windows
  • The source code is available on the github page
  • The module can be installed by $ python3 -m pip install browserhistory

kcp18microblog.com

Flask, MySQL with SQLAlechemy, and Linux Server

*Note
This website is a part of Miguel Grinberg 's flask tutorials. I emulated his design and implementations to build the website.

  • A web blog that uses the Flask web framework along with the flask extensions
  • It uses SQLAlchemy to store users and posts data in a MySQL database server
  • A virtual Linux server to run the web, supplemented by gunicorn, supervisor, and nginx softwares
  • Set up HTTPS encryption and acquire digital certificates by Let’s Encrypt
  • Please visit https://kcp18microblog.com to play around
